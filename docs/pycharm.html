A tutorial about how to connect Docker with Pycharm to write python code.

Steps to follow

Step1: Create a new project on Pycharm, choose a name of your preference.

Step2: Add a gitignore file and write .idea and venv.

Step3: create a new file name 'Dockerfile', next to the src folder not inside.

step4: type exactly what in the image below.

Step5:Open preference of pycharm, select build>Docker, this will make sure that docker is running and is successfully connect to pycharm. Click ok.

Step6: Click on Add configuration on top right corner. Click the "+" and select Docker.

Enter the name as "Barcode" and click the dropdown in dockerfile field and select Dockerfile.

Finally give the same name as "Barcode" tp the Image Tag.

Step7:Click green play sign on the top right corner and run the Dockerfile. It should look like the image below

Step8:We need to add the interpreter

Click on th project interpreter on the bottom left corner, to add Interpreter

Step9: Click on Docker and wait for it load the libraries, then Click "Apply" and click "ok"

Congratulations Ypu have now learn how to connect Pycharm with Docker.
