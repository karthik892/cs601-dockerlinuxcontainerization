A tutorial on how to setup a web server on your local computer using docker

Steps to follow

Step1: Make a folder on desktop name Docker or anything to your preference.

Step2: Open Docker and create simple code for PHP application "Hello World" with Echo.

Step3: Save that in a folder called "src" as source and we will name the file index.php

Step4: create a new file name 'Dockerfile' so we can host on the web server, next to the src folder not inside.

step5: type exactly what in the image below in order to get official php and run commands with it.

Step6:Open terminal or command line, insert docker build -t hello-world if current directory put . if not then put the directory name

Step7: Run the image by typing docker run -p 80:80 hello-world

Step8:Open Google Chrome and type in localhost and you will see "Hello World". See image below for reference.

Step9:Hit Control C to stop the container

step10:To mount a volume, type docker run -p -v /User/Name/Desktop/docker/src/:/var/www/html/ hello-world, now you can make changes in the local and they will reflect on web server

Congratulations, you have learned how to host a container on a web server!
